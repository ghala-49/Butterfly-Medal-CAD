A geometric butterfly medal designed and modeled in Onshape. The project focuses on creating a clean, structured, and visually balanced shape suitable for use as a decorative medal or accessory.

## Preview

![Butterfly Medal](butterfly%20medal.png)

## Overview

This model was created using Onshape’s 3D design tools, with attention to symmetry, smooth surfaces, and clear geometric divisions. The butterfly structure includes upper and lower wings, a segmented body, and simplified antennae to maintain a clean and modern look.

## Design Process

### Concept
The goal was to create a butterfly-shaped medal with a minimal, geometric style while keeping the form recognizable and visually appealing.

### Modeling
The model was built using Onshape features such as sketches, extrusions, fillets, and surface adjustments to achieve smooth reflective wings and a balanced overall structure.

### Final Output
The final design is suitable for rendering, 3D printing, or use as a digital asset. The clean geometry makes it adaptable for different materials and finishes.

## Notes

The project can be expanded by adding color variations, engraving details, or preparing the model for manufacturing.
